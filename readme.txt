To run the AI you will need to move the disired verion from the versions folder and then put it into the main folder directory

Installing Python on Windows:

Go to the Python official website at https://www.python.org/downloads/ and download the latest version of Python for Windows.
Run the installer and follow the instructions provided by the installation wizard.
During the installation, make sure to select the option to add Python to your PATH environment variable, so you can easily use Python from your terminal.
Once the installation is complete, you can test the installation by opening a command prompt or PowerShell and typing python. If Python is installed correctly, the version number should be displayed.

Installing Python on Linux:

Most Linux distributions come with Python pre-installed. You can check if Python is already installed on your system by opening a terminal and typing python --version.
If Python is not already installed, you can use your distribution's package manager to install it. For example, on Ubuntu or Debian-based systems, you can run sudo apt-get install python3 to install Python 3.
On Arch-based systems, you can run sudo pacman -S python to install Python 3.
Once Python is installed, you can test the installation by opening a terminal and typing python. If Python is installed correctly, the Python prompt should appear.

Installing Python on Mac OS:

Go to the Python official website at https://www.python.org/downloads/ and download the latest version of Python for Mac OS.
Run the installer and follow the instructions provided by the installation wizard.
During the installation, make sure to select the option to add Python to your PATH environment variable, so you can easily use Python from your terminal.
Once the installation is complete, you can test the installation by opening a terminal and typing python. If Python is installed correctly, the version number should be displayed.

1- Install the openai package by running pip install openai in your terminal.
2- Install the discord package by running pip install discord in your terminal.
3- Install the pytz package by running pip install pytz in your terminal.
4- Install the configparser package by running pip install configparser in your terminal.
5- Install the speech_recognition package by running pip install SpeechRecognition in your terminal.
6- Install the requests package by running pip install requests in your terminal.
7- Install the gtts package by running pip install gTTS in your terminal.
8- Install the playsound package by running pip install playsound in your terminal.

Note that some of these packages (such as os, time, and datetime) are built-in and do not need to be installed separately.
